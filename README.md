## 🔧 Setting Up Chirps-AI for Your Website

After installation, you will need to connect Chirps-AI with your website.

1. **Log In or Register**

   The app may ask you to log in or create an account. This step links your AI agent with your preferences.

2. **Get the Widget Code**

   Chirps-AI provides a small piece of code called a widget script. This code embeds the AI agent into your website.

3. **Add Code to Your Website**

   - For WordPress: Go to the “Appearance” menu, select “Widgets,” and add a Custom HTML widget. Paste the code there.
   - For Shopify: Go to “Online Store” > “Themes” > “Actions” > “Edit code” and paste the code in the theme’s footer file.
   - For other platforms: Look for the option to add custom HTML or scripts in your website builder and paste the code.

4. **Save and Publish**

   Save your changes and publish your website.

5. **Test the Widget**

   Visit your website and try the AI agent. It should respond to voice calls and live chat instantly.
